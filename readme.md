| 声明方式     | 用途       | 接收数据位置                              | 说明                                 |
|--------------|------------|-----------------------------------------|------------------------------------|
| Query(...)   | 查询参数   | URL 查询字符串                          | 用于 GET 请求的查询参数             |
| Path(...)    | 路径参数   | URL 路径                               | 用于路由路径中的参数，比如 /items/{id} |
| Header(...)  | 请求头参数 | HTTP 请求头                           | 用于获取请求头信息                 |
| Cookie(...)  | Cookie 参数| HTTP Cookie                          | 获取客户端 Cookie                  |
| Body(...)    | 请求体参数 | HTTP 请求体（通常是 JSON）             | 用于接收 JSON、复杂结构请求体       |
| Form(...)    | 表单数据   | HTTP 请求体（x-www-form-urlencoded 或 multipart/form-data） | 用于接收表单格式数据               |
| File(...)    | 文件上传   | HTTP 请求体（multipart/form-data）    | 用于接收上传文件                   |
| Depends(...) | 依赖注入   | 任何（由依赖函数决定）                  | 用于声明依赖注入，常用于权限验证、数据库连接等 |


方法	用途	说明
GET	查询资源	获取资源，安全且幂等
POST	创建资源	新建资源，不幂等
PUT	更新资源	全量更新，幂等
PATCH	部分更新资源	局部修改，不一定幂等
DELETE	删除资源	删除指定资源，幂等
